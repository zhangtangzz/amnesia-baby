# 一、模块目标

素材解析系统（Material Parser）负责将用户上传的原始素材转换为可计算、可检索、可推理的结构化数据。

支持：

* 视频
* 音频
* 文字稿
* 文章
* 网页资料
* 社交媒体内容

输出：

* 人格证据
* 知识证据
* 行为证据
* 价值观证据
* 语音特征

供后续模块使用。

---

# 二、系统定位

素材解析系统不负责：

* 人格分类
* 九型人格推理
* 大五人格计算
* 行为预测

素材解析系统只负责：

```text
发现证据
整理证据
结构化证据
```

# 六、视频解析规范

## Step1 视频转文字

目标：

提取角色说过的话。

输出：

```json
{
  "timestamp": "15:32",
  "content": "创业最大的风险是不创业"
}
```

---

## Step2 情绪识别

识别：

* 开心
* 兴奋
* 愤怒
* 悲伤
* 平静

输出：

```json
{
  "emotion": "excited",
  "confidence": 0.87
}
```

---

## Step3 行为分析

识别：

* 肢体动作
* 表情
* 互动方式

输出：

```json
{
  "behavior": "积极表达",
  "confidence": 0.82
}
```

---

## Step4 人格证据提取

示例：

原文：

```text
我喜欢挑战不可能
```

输出：

```json
{
  "trait": "risk_preference",
  "evidence": "喜欢挑战不可能",
  "confidence": 0.84
}
```

# 十四、证据可信度

所有输出必须包含：

```json
{
  "confidence": 0.87
}
```

---

## 评分标准

| 分数      | 含义   |
| ------- | ---- |
| 0.9-1.0 | 高可信  |
| 0.7-0.9 | 较可信  |
| 0.5-0.7 | 一般可信 |
| <0.5    | 低可信  |

# V1核心结论

素材解析系统负责回答：

```text
素材中有什么信息？
```

人格建模负责回答：

```text
这些信息说明他是什么样的人？
```

知识库负责回答：

```text
这些信息说明他知道什么？
```

行为预测负责回答：

```text
在新的场景下他会怎么做？
```

素材解析系统是整个《失忆宝宝》数字人格重建流程的入口，也是所有推理链路的数据基础。
