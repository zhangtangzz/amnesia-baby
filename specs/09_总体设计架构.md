# 一、项目目标

失忆宝宝的目标不是构建聊天机器人。

而是：

构建可持续成长的数字人格系统（Digital Personality System）。

系统能够根据角色素材：

* 重建人格
* 重建知识
* 重建行为模式
* 重建表达方式

最终实现：

```text
给AI一个人
让AI成为这个人
```

# 五、系统分层

## 第一层：素材层

负责接收：

```text
视频

音频

文字稿

文章

网页资料

社交媒体
```

输入：

```text
Raw Material
```

输出：

```text
Structured Evidence
```

---

## 第二层：素材解析层

对应：

```text
Material Agent
```

负责：

```text
提取人格证据

提取知识证据

提取行为证据

提取价值观证据

提取语音特征
```

输出：

```json
{
  "personality_evidence": [],
  "knowledge_evidence": [],
  "behavior_evidence": [],
  "value_evidence": []
}
```
