# 人格成长规范（V1）

## 文档目标

定义《失忆宝宝》人格成长系统。

人格成长系统负责：

* 接收新的角色素材
* 接收新的角色行为
* 接收新的对话记录
* 修正人格模型
* 更新知识体系
* 记录人格演化过程

最终实现：

```text id="uxu1hy"
人格不是固定标签

而是持续成长的数字人格
```

---

# 一、为什么需要人格成长

现实中的人不会永远保持不变。

例如：

```text id="s55j9v"
20岁

冒险
冲动
激进
```

↓

```text id="3bgxwn"
40岁

谨慎
成熟
保守
```

人格会变化。

价值观会变化。

行为也会变化。

---

如果系统永远保持：

```text id="2z3dvh"
第一次建模结果
```

那么：

```text id="pp7exv"
人格会逐渐失真
```

---

# 二、成长目标

人格成长系统需要回答：

```text id="vhjml9"
角色变了吗？

变了什么？

为什么变化？

变化是否可信？
```

---

```text id="3xxc8y"
新自传
```

### 示例

例如：

过去：




```text id="1p6m24"

```

# 十五、V1核心结论

人格建模负责：

```text id="4r8yde"
角色现在是什么样的人
```

人格成长负责：

```text id="5jwph9"
角色是如何变成这样的人
```

成长系统回答三个问题：

```text id="p8psbi"
变了吗？

为什么变？

变化是否可信？
```

最终形成：

```text id="1jlwm7"
静态人格
+
动态成长
+
时间轴记录
```

构建真正可持续演化的数字人格。
